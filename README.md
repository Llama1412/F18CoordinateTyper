#   LCTRL+NUM1-4    LEFT MDI PB 6-9                         #
#   LCTRL+NUM6      LEFT MDI PB 18                          #
#   LCTRL+NUM7      LEFT MDI PB 4                           #
#   LCTRL+NUM8      LEFT MDI PB 11                          #
